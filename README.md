# The Biryani Box Elite Web-App 🍛

Premium Cloud Kitchen Web-App with AI Agent Integration.

## Features
- **Royal Green & Gold Theme:** Mobile-responsive premium UI.
- **Groq AI Agent:** Personalized daily greetings and menu suggestions.
- **Loyalty Wallet:** ₹10 reward for IG Story sharing.
- **Interactive Menu:** SEO-optimized keywords and items.

## Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Backend:** Python (Flask) on Vercel Functions
- **AI:** Groq Llama-3 API

## Deployment
1. Connect this repo to Vercel.
2. Add `GROQ_API_KEY` in Environment Variables.
3. Deploy!
